Free Responsive HTML5 Template

Kavin corporate Bootstrap Responsive Web Template is for multi-purpose use it is a clean html5 template built with valid HTML5 & CSS3. It's based on latest Bootstrap framework 3.3.1 fully responsive web compatible with multi browser and devices. This template can be used for multi-purpose needs like business, consultancy, agency, personal portfolio, profile, mobile website and start-up company.
